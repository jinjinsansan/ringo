# 「りんご会♪」UI/UXデザイン仕様書

**サイト名**：りんご会♪  
**コンセプト**：柔らかい、女性向け、可愛い、親しみやすい  
**ターゲット**：20代～40代女性

---

## 1. 全体的なデザイン方針

### カラーパレット

| 色 | カラーコード | 用途 |
|----|-----------|------|
| メインカラー（ピンク） | #FFB6D9 | ボタン、ヘッダー、アクセント |
| サブカラー（薄紫） | #E6D4F0 | 背景、セクション分け |
| アクセントカラー（赤りんご） | #E63946 | 重要な情報、警告 |
| テキスト（濃いグレー） | #333333 | 本文テキスト |
| 背景（白） | #FFFFFF | ページ背景 |
| 薄い背景 | #FFF5F9 | 背景パターン |

### フォント

| 用途 | フォント | 備考 |
|------|---------|------|
| ロゴ・タイトル | 「ふんわり」「丸ゴシック」 | 可愛らしい印象 |
| 本文 | 「ヒラギノ角ゴ」「Segoe UI」 | 読みやすい |
| 数字 | 「Roboto」 | 統一感 |

### デザイン要素

- **丸みのあるボタン**：角度は15px以上のボーダーラジウス
- **グラデーション**：ピンク～紫のグラデーション背景
- **アイコン**：可愛らしいシンプルなアイコン
- **イラスト**：手書き風、温かみのあるイラスト

### フォーム要素

**メールアドレス入力欄**
```
- ラベル：「メールアドレス」
- プレースホルダー：「example@example.com」
- 入力型：email
- バリデーション：メール形式の確認
- エラーメッセージ：「正しいメールアドレスを入力してください」
```

**パスワード入力欄**
```
- ラベル：「パスワード」
- プレースホルダー：「8文字以上の英数字」
- 入力型：password
- バリデーション：8文字以上、英数字を含む
- エラーメッセージ：「パスワードは8文字以上の英数字で入力してください」
- パスワード強度インジケーター：弱い/中程度/強い
```

**パスワード確認入力欄**
```
- ラベル：「パスワード確認」
- プレースホルダー：「パスワードを再度入力」
- 入力型：password
- バリデーション：パスワードと一致
- エラーメッセージ：「パスワードが一致しません」
```

**利用規約チェックボックス**
```
- □ 利用規約に同意する
- リンク：「利用規約」をクリックで利用規約ページに遷移
```

**登録ボタン**
```
- テキスト：「登録する」
- 背景色：ピンク（#FFB6D9）
- 幅：100%
- ホバー時：背景色を濃くする
- クリック時：ローディングアニメーション表示
```

## 14. アクセシビリティ

### 対応

- **色覚異常対応**：色だけで情報を伝えない（テキスト、アイコン併用）
- **スクリーンリーダー対応**：適切なALT属性、ARIA属性
- **キーボード操作対応**：Tab キーでナビゲーション可能
- **コントラスト**：WCAG AA 準拠
